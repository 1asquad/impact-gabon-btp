📦 Comment intégrer

Option recommandée : remplacer le bloc :root { ... } dans styles.css par le contenu de
styles-refined-v4.css (tout le reste du fichier CSS reste inchangé).

Les alias sémantiques (--blue, --green, etc.) pointent vers les nouvelles valeurs via var() :
--blue: var(--blue-500); ← tout le code existant continue de fonctionner sans modification.
